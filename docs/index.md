  A full walkthrough of what TezQR does, who uses it, and how the major flows work.

- [Architecture Guide](architecture.md)

- [FastAPI Backend Learning Roadmap](learning-roadmap.md)

1. [Product Tutorial](product-tutorial.md)
2. [Architecture Guide](architecture.md)
3. [Deployment Guide](deployment.md)

If you want to learn backend engineering from the repo, read this order:

1. [FastAPI Backend Learning Roadmap](learning-roadmap.md)
2. [Product Tutorial](product-tutorial.md)
3. [Architecture Guide](architecture.md)
